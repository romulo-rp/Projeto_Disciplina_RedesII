# 🌐 Projeto de Infraestrutura de Redes – P2 Marketing

Este projeto consiste no desenvolvimento completo de uma infraestrutura de rede estruturada para um ambiente corporativo, seguindo boas práticas e normas técnicas.

## 🎯 Objetivo
Planejar e estruturar uma rede eficiente para uma empresa de marketing, garantindo conectividade, organização e desempenho.

## 🏢 Cenário
Empresa com dois andares contendo setores como atendimento, pesquisa, planejamento, mídia, criação e produção, além de áreas comuns.

## ⚙️ Tecnologias e Ferramentas
- Cisco Packet Tracer
- Conceitos de Redes de Computadores
- Norma NBR 14565
- Cabeamento estruturado (EIA/TIA 568A)

## 🧩 Funcionalidades do Projeto
- Definição da topologia física e lógica da rede
- Planejamento de rede cabeada e Wi-Fi
- Dimensionamento de equipamentos (switches, roteadores, racks)
- Organização de cabeamento estruturado
- Elaboração de orçamento completo
- Identificação e padronização dos pontos de rede
